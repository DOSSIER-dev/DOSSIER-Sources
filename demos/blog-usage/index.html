How To Use

Quickstart

Just include the library in the head or body section of the page: 
          <script type="text/javascript" src="https://localhost/embed/lib/sources.js"></script>

'Sourcebox' and mouseover-panel are disabled by default, but can be enabled in the options. Options can be set in the following manner: 

      <script>
        window.sourcesjs = {

          // Render an element with a list of all sources found on the page
          linkbox: false,

          // Display an info-box when hovering over a source link
          hoverbox: false,

          // The hostname of the sources installation
          appserver: 'https://localhost',

          // A css-class that is added to a source link
          linkClassName: 'sourcesjs-parsed',

          // Add the sources svg icon to source links
          linkIcon: true
        };
      </script>

If enabled, the source box will be appended into an element with it 'sourcesjs-box', e.g. 

        ...
        <div id="sourcesjs-box"></div>
        ...

If such an element does not exist, the source box gets appended to document body.
